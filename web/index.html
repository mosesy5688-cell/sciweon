Design principles

  • Agent-first Designed for machines to consume directly. No human search-UX overhead in the data layer.
  • Cross-source Same fact verified across independent providers. Disagreement is captured, not hidden.
  • Provenance Every value carries its source, timestamp, and extraction method. Auditable end to end.
  • Confidence quantified Single-source values capped. Multi-source consensus boosts confidence. Agents see the score.
  • Negative evidence Failed trials, retractions, withdrawn drugs, post-market signals - collected as first-class data.
  • Daily-updated Continuously refreshed. Historical versions retained so agents can reason over time, not just snapshots.

For AI agents

Sciweon is not a search engine for humans. The intended user is an AI agent doing real research work - selecting compound leads, scoring trial designs, avoiding prior-failure repeats, choosing the right model for a domain task.
